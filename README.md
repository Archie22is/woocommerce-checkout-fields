## WooCommerce Checkout Fields

This plugin provides an implementation of  [WooCommerce](https://docs.woothemes.com/document/tutorial-customising-checkout-fields-using-actions-and-filters/) for checkout fields.

With this plugin you can easily manipulate checkout fields, adds new fields, delete fields, change class, use it to validate fields.
